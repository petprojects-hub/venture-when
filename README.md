# Venture When

A full-stack application with Angular frontend and Node.js backend.

## Project Structure

- `frontend/` - Angular application
- `backend/` - Node.js API server

## Getting Started

### Frontend
```bash
cd frontend
npm install
ng serve
```

### Backend
```bash
cd backend
npm install
npm start
```

## Features

- Angular frontend with routing and SCSS
- Node.js backend with Express
